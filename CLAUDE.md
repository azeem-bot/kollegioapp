# Kollegio Picker Demo
Stack: Vite + React + TypeScript, pure CSS with custom properties, no UI library.
Font fallback: Inter from Google Fonts (production uses ABC Diatype Rounded Bold).
Design tokens: --bg-base:#f9f7f2, --bg-white:#fff, --content-primary:#132619, --content-secondary:rgba(19,38,25,0.65), --border-primary:rgba(19,38,25,0.3), --bg-g1:#132619, --bg-g2:#227322, --bg-g3-10:rgba(119,204,91,0.1)
No hardcoded data inside components. All options, titles, and modes come from props.
